Projects

Below are a list and a brief description of various projects that I have worked on over the last six years.

RPS-LICENSING - Licensing is a large work flow management application built to handle 1500-1800 users at the Nuclear Regulatory Commission as part of the larger Reactor Programming System it was built using C#, Asp.Net and Entity Framework on the backend and Angular, Bootstrap and other various libraries on the frontend.
RPS - OL - OL is an application processing system that is part of the NRC’s RPS project. It is used to help manage tens of thousands of nuclear reactor operators and potential operators. It’s built using Asp.Net/Angular with special attention paid to the responsiveness of the application.
Operations Xchange - OX is a mobile organizational communications platform originally targeted at helping college football teams communicate with their players and coaching staffs. During its lifespan it has seen major architecture upgrades including a switch from a jQuery heavy application to an Angular application. OX was designed to run natively or responsively on a phone or iPad and Apache Cordova / PhoneGap has been used to help allow for operation on devices natively.
Tyler Robinson Foundation - TFR is the foundation founded by Imagine Dragons as they seek to help ‘Slay Cancer’. The TFR website was designed for a great mobile experience. It was built using WordPress, Bootstrap and JQuery.
Toni’s Kiddie Korner - Toni’s Kiddie Korner is a local preschool who contracted me to help design and develop a simple website for their customers use. It uses bootstrap and basic HTML5 and CSS to help achieve a simple, responsive and fun design.
Henry Jackson Foundation - HJF - HJF is a data and accounting reconciliation tool built using C#, JQuery and Asp.net MVC.
ISOEZ - A future SASS offering built using heavy amounts of jQuery and vanilla JS.
OPC Link - OPC Link is an assembly line management solution built for GE. It was built using C# Asp.net and a JavaScript frontend.
ISOHelpers.com - WordPress marketing website utilizes bootstrap, jQuery and PHP.
Novaqual.com - WordPress marketing website utilizes bootstrap, jQuery and PHP.
